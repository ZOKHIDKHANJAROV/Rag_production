🔴 "Sorry, I encountered an error. Please try again."

This is the most common error. See solutions →

Quick fix: Run diagnostic script:

bash diagnostic.sh # or .\diagnostic.ps1

🔴 "Cannot connect to RAG service"

See solutions →

Quick fix: Check .env configuration and service URLs

🔴 Service timeout (504 error)

See solutions →

Quick fix: Increase REQUEST_TIMEOUT in .env

🔴 Port already in use

See solutions →

Quick fix: Kill process or use different port

🔴 File upload fails

See solutions →

Quick fix: Check file size limit in .env

🚀 Getting Started in 3 Steps

Step 1: Start Services

cd /path/to/rag-production
docker-compose up -d

Step 2: Verify Setup

bash diagnostic.sh
# or .\diagnostic.ps1 on Windows

Step 3: Access Web Interface

Open your browser: http://localhost:8000

Done! Now create a new chat and ask your first question. See full setup guide →
